# ColorStack Livingstone Website
Website for the ColorStack Livingstone chapter at Livingstone College. It highlights the community, member spotlight, and ways to get involved.

## What it stands for

ColorStack Livingstone stands for empowering Black excellence in tech at Livingstone College.

- "Color" represents identity, visibility, and inclusion for Black students in computing.
- "Stack" represents the full technical stack and the layered support system: skills, mentorship, and opportunity.
- "Livingstone" anchors the mission to the Livingstone College campus community in Salisbury, NC.

## Site sections

- Hero welcome and mission statement.
- Member Spotlight slideshow with achievements.
- Get Involved cards for sponsorship, joining, and contact.
- Footer with quick links and social channels.

## Features

- Responsive navigation with mobile toggle.
- Smooth scroll and active nav highlighting.
- Spotlight carousel with auto-rotation and manual controls.
- Decorative UI elements and hover effects.

## Tech stack

- HTML, CSS, and vanilla JavaScript.
- No build tools or dependencies.

## Run locally

- Open `index.html` directly in your browser, or
- Serve the folder with any static server (for example, `python -m http.server 8000`) and visit `http://localhost:8000`.
